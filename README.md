# Weekly Todo Calendar

An Obsidian calendar and Todo workflow built around one Markdown note per week.

Weekly notes open directly as an editable Monday–Sunday calendar. Timed tasks remain ordinary Markdown, and macOS users can sync them to an Apple Calendar that is shared through iCloud with iPhone, iPad, and other Macs.

## Features

- Open weekly Todo notes directly in a full-week calendar.
- Drag, resize, create, complete, and edit timed tasks in the calendar.
- Keep the calendar source as portable Markdown.
- Use compact week titles such as `0622-0628`.
- Display tasks from daily notes, Obsidian Tasks dates, and ICS calendars.
- Sync a weekly note to an existing Apple Calendar on macOS.
- Update previously synced events instead of creating duplicates.

## Apple Calendar and iCloud

Apple Calendar export is available in the Obsidian desktop app on macOS. Choose an existing iCloud-backed Apple Calendar in plugin settings, then run **Weekly Todo Calendar: Sync current week to Apple Calendar**.

The calendar view itself works on mobile, but the Apple Calendar export command requires macOS.
